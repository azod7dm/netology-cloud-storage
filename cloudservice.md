# Cloud Storage BackendБэкенд для облачного хранилища, реализованный на Spring Boot.## Стек- Java 17- Spring Boot 3- PostgreSQL- Docker## Запуск1. Соберите: `mvn clean package`2. Запустите: `docker-compose up --build`## АвторизацияИспользуйте токен:## API- `POST /file` — загрузка- `GET /list?limit=N` — список- `DELETE /file?filename=...` — удаление- `PUT /file?filename=old` + `{ "newname": "new" }` — переименование- `GET /file?filename=...` — скачивание## Архитектура- Контроллеры обрабатывают HTTP- Сервисы содержат бизнес-логику- Репозитории работают с БД- Логгирование через SLF4J
